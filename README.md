## 📖 О проекте

**AutoServiceDetailPro** — это не просто учётная система. Это **экосистема для детейлинг-центров**, которая превращает хаос из заказов, полиролей и клиентов в идеальный порядок.

> ⚡ *"От мойки двигателя до керамики — каждый этап под контролем."*

### 🎯 Для кого?
- 🧽 Владельцы детейлинг-студий
- 🧴 Мастера-полировщики и установщики плёнок
- 📊 Менеджеры, уставшие от Excel и записочек на стикерах
